# Description

A library of 4 simple cipher algorithms:

* Caesar's cipher
* Spartan's cipher
* Vigenere's cipher
* One-time pad (OTP) cipher

## Compile

Use the provided Makefile:

### Building the library

```bash
make crypto.o
```

### Building the test files

Using the library is demonstrated in [main.c](main.c)

* Build the tests:

```bash
make crypto
```

* Run the tests:

```bash
./crypto
```
