# Cipher algorithms

A library of 4 simple cipher algorithms:

* Caesar's cipher
* Spartan's cipher
* Vigenere's cipher
* One-time pad (OTP) cipher

## Compile

Build the library:

```bash
make crypto.o
```

## Tests

Using the library is demonstrated in [main.c](main.c)

Build:

```bash
make crypto
```

Run:

```bash
./crypto
```
